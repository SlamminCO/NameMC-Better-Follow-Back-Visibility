# NameMC Better Follow Back Visibility
 A simple chrome extension that improves the follow back status visibility on NameMC
 
 By default, NameMC only shows followback status by whether or not the user's name is bold, but with this extension enabled, the following page will show a clear indicator of whether or not someone follows you back

# Installation
 1. Click the green Code button and select Download Zip
 <img width="396" alt="image" src="https://user-images.githubusercontent.com/100872889/156704401-a45064d6-5907-494b-8924-0b7d1a34ca50.png">
 
 2. Extract the zip file.
 3. In your browser address bar, go to chrome://extensions
 <img width="1206" alt="image" src="https://user-images.githubusercontent.com/100872889/156704955-824cdce8-6c6a-4560-9a4c-0069918567d6.png">

 4. Enable Developer Mode
 <img width="205" alt="image" src="https://user-images.githubusercontent.com/100872889/156705220-b5647c59-bab0-4e93-9c14-5c563798cc5d.png">

 5. Click Load Unpacked
 <img width="370" alt="image" src="https://user-images.githubusercontent.com/100872889/156705301-678843ca-8292-4c30-a95f-9b4017840c17.png">

 6. Finally, select the unzipped folder
 <img width="292" alt="image" src="https://user-images.githubusercontent.com/100872889/156705651-aeae1667-1b53-4179-8929-920b348ef566.png">
